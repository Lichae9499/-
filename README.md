# ADHD 自测问卷

专业的注意力缺陷/多动障碍评估工具

## 功能特点

- 🧑 成人版 ADHD 测试 (ASRS-v1.1)
- 👶 儿童版 ADHD 测试 (Vanderbilt评定量表)
- 📊 详细的评分和分析
- 💡 专业建议和指导

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到 Vercel

1. 将项目推送到 GitHub
2. 在 Vercel 中导入 GitHub 仓库
3. Vercel 会自动检测配置并部署

## 技术栈

- React 18
- Vite
- Tailwind CSS
- Lucide React (图标)
